🎓 From TA recordings — exam strategy

📌 Only what Raskolnikov says in class is tested. If it's in the reading but not said in class, it is not on the exam. If he says "this won't be on the exam," skip it.

📌 Attack outline strategy: Run through code sections like a checklist. For in-kind benefits: §119 → check → §132 → check → conclusion. One sentence per issue. Cite the code.

📌 Word problems: one sentence per issue. "This qualifies as X because Y — §Z." That is the level of detail he wants.

📌 MC is substantive, not memory-based. He is not testing whether you memorized which court said what.

📌 Bring a basic 4-function calculator. No phones. ~$10. You will need it.
